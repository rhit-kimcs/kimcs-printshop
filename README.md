# Printing Capstone Project
This is a new implementation of the Print and Copy Order Form for the Rose-Hulman CSSE Department.

## Setup
Create a file called `.env` within the `print-backend` folder. Within that file paste the following code, and change the username and password as specified.
```
DB_USER = yourusername
DB_PASSWORD = Password123
DB_HOST = golem.csse.rose-hulman.edu
DB_PORT = 1433
DB_NAME = PrintShop
```

## Running the Project
Within the main repository, run `pnpm dev`
